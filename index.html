Colour Table

This project is currently being revisited, and so colours will likely go missing and/or change.

This table is a collection of colours you may want to use in your Satisfactory builds, based on in-game items, resources, UI elements, etc.

Click any colour to copy its hex code to your clipboard, and use CTRL + V to paste it into your Customiser.
